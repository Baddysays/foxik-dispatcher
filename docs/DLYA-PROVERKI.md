# Для ревью (MOX)

**Демо:** https://foxik.baddysays.ru  
**Репозиторий:** см. README в корне проекта

## Быстрая проверка

1. Откройте демо (или локально: `npm install` → `npm run dev`)
2. **Обзор** — KPI, карта, хронология (имя при наведении), топ-3, 5 вопросов ТЗ
3. **Статистика** — окно замеров, веса формулы, фильтры, редактирование заметности в таблице
4. Подвиньте «Добыча» — рейтинг пересчитается
5. Окно «Ночное время» — дневные наблюдения выпадут из отчёта
6. **Добавить** → «Сброс к данным ТЗ» → 5 наблюдений MOX, лидер Рыжик ≈ 12.40
7. **Добавить** → JSON — загрузите файлы из `docs/samples/` (см. README там)
8. **AI Worklog** — 6 чекпоинтов работы с AI

## Тестовые JSON (`docs/samples/`)

| Файл | Что проверяет |
|------|----------------|
| `01-baseline-mox.json` | Эталон ТЗ, 5 obs, лидер fox_001 ≈ 12.40 |

| `03-mixed-validation.json` | Пропуски (локация/время), «неизвестен» окрас, предупреждения |
