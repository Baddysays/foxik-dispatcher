# Тестовое задание | AI-first Developer в MOX

## Коротко
- Срок: до 5 дней
- Сценарий: «Лисий диспетчер» (выбран) или «Ферма невидимых кроликов»
- Интерактивный UI + AI Worklog (5-7 чекпоинтов)
- Менять данные/параметры и видеть пересчет

## Вариант 01 — Лисий диспетчер

Стартовый JSON (дублируется в src/data.js):

```json
[
  {"id":"obs_001","fox_id":"fox_001","location":"Северная поляна","color":"рыжая","has_prey":true,"suspicion_level":8,"time":"08:20"},
  {"id":"obs_002","fox_id":"fox_002","location":"Туманная тропа","color":"черная","has_prey":false,"suspicion_level":5,"time":"09:05"},
  {"id":"obs_003","fox_id":"fox_001","location":"Северная поляна","color":"рыжая","has_prey":false,"suspicion_level":9,"time":"10:40"}
]
```

Источник: `docs/official-test-task.html` (официальный файл MOX).

### Из интерфейса должно быть понятно
- Сколько уникальных лис
- Где основная активность
- Что влияет на подозрительность
- Какая лиса самая подозрительная и почему
- Как меняется отчет при изменении данных

## AI Worklog
5-7 чекпоинтов в UI. Без секретов и токенов.

## Формат сдачи
1. Ссылка на интерфейс
2. GitHub или архив
3. README
4. AI-инструменты и проверка
5. Worklog в интерфейсе
